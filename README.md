## 🎯 Sobre o Projeto

O **Landing Page Business** é um template de landing page profissional, construído com foco em **conversão de leads** e **experiência do usuário**. Embora a versão de demonstração apresente um cenário para eletricista/segurança residencial, toda a estrutura foi pensada para ser **facilmente adaptável a qualquer tipo de negócio**, como:

- 🔧 Encanadores, pintores e técnicos
- 🏥 Clínicas e consultórios
- 🍕 Restaurantes e delivery
- 🏋️ Academias e personal trainers
- 💈 Barbearias e salões de beleza
- 📸 Fotógrafos e videomakers

### Por que este template?

| Benefício | Descrição |
|---|---|
| 🚀 **Alta Performance** | Construído com Vite para carregamento instantâneo |
| 📱 **Totalmente Responsivo** | Adapta-se perfeitamente a mobile, tablet e desktop |
| 🎨 **Design Premium** | Dark mode com gradientes, glassmorphism e micro-animações |
| 💬 **Integração WhatsApp** | Botão flutuante e CTAs com link direto para o WhatsApp |
| 🔍 **SEO Otimizado** | Meta tags, Open Graph e estrutura semântica |
| ♿ **Acessível** | Desenvolvido seguindo boas práticas de acessibilidade |

## 🚀 Instalação e Uso

```bash
# 1. Clone o repositório
git clone https://github.com/DaviSantos040910/Landing-Page-Business.git

# 2. Acesse a pasta do projeto
cd Landing-Page-Business

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

O projeto estará disponível em `http://localhost:5173`

## 🎨 Personalização

O template foi projetado para fácil customização. Cada componente possui comentários `{/* EDITE: */}` indicando os pontos de personalização:

### 1. Informações do Negócio

Edite os seguintes arquivos para trocar os dados:

| Arquivo | O que personalizar |
|---|---|
| `src/components/HeroSection.tsx` | Título, subtítulo, estatísticas e CTAs |
| `src/components/ServicesSection.tsx` | Lista de serviços e descrições |
| `src/components/AboutSection.tsx` | Foto, nome e descrição profissional |
| `src/components/ContactSection.tsx` | Telefone, e-mail, endereço e horário |
| `src/components/WhatsAppButton.tsx` | Número do WhatsApp |
| `src/components/Footer.tsx` | Nome, redes sociais e cidade |

### 2. Identidade Visual

- **Cores**: Edite as variáveis CSS em `src/index.css`
- **Fontes**: Atualize a família de fontes em `tailwind.config.ts`
- **Imagens**: Substitua os arquivos na pasta `public/`

### 3. SEO e Meta Tags

- Atualize o `<title>` e meta tags em `index.html`
- Configure o `public/robots.txt` com o domínio correto

### 4. Integração com Resend (E-mails)

Para que o formulário de contato envie e-mails, você precisa configurar o [Resend](https://resend.com).

1.  Crie uma conta gratuita no Resend.
2.  Gere uma **API Key**.
3.  Configure a variável de ambiente `RESEND_API_KEY` na sua plataforma de hospedagem (Vercel, Netlify, etc.).
4.  No arquivo `api/contact.ts`, verifique se o e-mail em `from` e `to` está correto.
